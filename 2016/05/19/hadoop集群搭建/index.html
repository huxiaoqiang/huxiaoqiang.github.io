Oozie系统研究（二）——搭建hadoop集群

发表于 2016-05-19

博客说明

近期导师要我研究一下大数据系统引擎oozie中工作流模式，也是我第一次接触到大数据相关的内容，在整个学习过程和实验环境的搭建过程中，学习到了许多知识，也遇到了诸多坑。想把这些经验和教训做个记录总结， 方便后续自己复习和后来人参考，如有错误之处，欢迎批评指正。
上一篇博客简要介绍了oozie，这篇博客开始介绍从0开始搭建hadoop集群。

Oozie环境依赖

  • Unix box (tested on Mac OS X and Linux)
  • Java JDK 1.6+
  • Maven 3.0.1+
  • Hadoop 0.20.2+
  • Pig 0.7+
  • ExtJS library (optional, to enable Oozie webconsole)

由于oozie对于依赖软件的版本要求很严格，最新版hadoop2.7.2尚未找到有人成功适配Oozie4.2.0，所以我选择的是hadoop2.6.0。java最新版是1.8，但是在尝试用1.8配置环境过程中，会因为版本问题出现bug，所以退回1.7，完美运行。ExtJS Library只允许用2.2版本，其他版本都不行。所以本次实验的环境如下：

  • Ubuntu 14.04 LTS
  • Java JDK 1.7
  • Hadoop 2.6.0
  • ExtJS library 2.2
  • Maven 3.3.9
